## easyAI

Visit [easy-ai-alpha.vercel.app](https://easy-ai-alpha.vercel.app) to use!

# Project outline

easyAI is a basic AI chatbot. Messages are free and infinite, rate limiting is high though so you have to wait a long time especially during periods of high usage for it to respond. I made this as a short quick experimental project and I will continue to add features so it should get better.

# Features:

- Free chat
- Infinite messages
- Bot can view past 10 messages as reference
- Bot can tell time
- Bot knows what day it is
- Bot randomly seems to recommend a gaming website... it must be amazing for the bot to like it so much
